    What is data science?

    Data science is a collection of techniques used to extract value from data.

    What are the different classes of data science problems?

    There are two major classes of data science problems. 1. supervised learning model 2. unsupervised learning model

    What is a supervised learning model?

    This model gives training to the machine for solving problems.

    What is an unsupervised learning model?

    This model does not give training to the machine for solving problems.

    What is deep learning?

    It is a machine learning model based on artificial neural networks, which are inspired by the functioning of human brain.

    What is data exploration?

    Data exploration is the detailed analysis of data to gain better understanding of it. It can be classified into 2 types. 1. Descriptive Statistics 2. Data Visualisation

    What is meant by descriptive statistics?

    It is the process of summarising the data using various statistical measures such as mean, median, mode etc.

    What is data visualisation?

    It is the process of making visual representation of data using various charts like histogram, scatter plot, bubble chart etc.

    When can we say that a machine has the ability to learn?

    A machine is able to learn if it is able to utilise an experience so that its performance improves on similar experiences in future.

    What are the types of machine learning algorithms?

    Machine learning algorithms can be classified into 3 types. 1. Predictive Model 2. Descriptive Model 3. Meta learning Model

    What is predictive model?

    In this model, we predict the value of a target variable based on a set of input variables Example - Predicting whether a football team will win or lose

    What is descriptive model?

    In this model, we identify useful patterns / associations within data. Example - Identify the items which are bought together from a shop

    What is meta learning model?

    This model deals with learning how to learn more effectively. This can be used for improving the performance of machine learning algorithms.

    What is a lazy learning approach?

    This is a learning approach in which classification is done only after preparing the test data.This approach spends less time on training, and more time on predicting.

    What is probabilistic learning?

    In probabilistic learning, the probabilities associated with various events are used for learning and prediction.

    What is decision tree learning?

    It is a machine learning technique used for classification and prediction. This uses a flow chart like tree structure called decision tree.

    What is classification rules learning?

    Classification Rules Learning is a machine learning approach based on classification rules.A classification rule represent knowledge in the form of logical if..else statements.This knowledge can be used for classification and prediction.

    What is Regression?

    Regression is a statistical technique used for estimating relationships between numerical data.

    What is simple linear regression?

    Simple Linear Regression finds out the linear relationship between single numeric depen- dent variable and a single numeric independent variable.

    What is multiple linear regression?

    Multiple Linear Regression finds out the linear relationship between single numeric dependent variable and multiple numeric independent variables.
